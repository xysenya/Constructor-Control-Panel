# Constructor: Control Panel

Панель управления для мастера НРИ (DnD), включающая интерактивную карту, таймер, аудиоплеер и синтезатор речи.

## Возможности

*   **Интерактивная карта:** Сетка 7x7 с возможностью редактирования ячеек, перемещения и детального просмотра.
*   **Таймер:** Обратный отсчет с визуальными эффектами (краснение, мигание) и звуковыми сигналами. Отдельное окно для вывода игрокам (второй монитор).
*   **Аудиоплеер:** Воспроизведение музыки и звуковых эффектов.
*   **Синтезатор речи:** Генерация голоса (Text-to-Speech) с использованием Microsoft Azure Neural Voices (через `edge-tts`). Возможность изменения тона и сохранения файлов.
*   **Игровые вкладки:** Карты уровней, списки предметов, листы персонажей.

## Установка и запуск

### Из исходного кода

1.  Установите Python 3.10+.
2.  Установите зависимости:
    ```bash
    pip install PyQt6 edge-tts
    ```
3.  Запустите программу:
    ```bash
    python main.py
    ```

### Сборка EXE

Для сборки в единый файл используется PyInstaller:

```bash
pyinstaller --noconfirm --onedir --windowed --icon "icon.ico" --name "ConstructorControlPanel" --collect-all edge_tts main.py
```

Для создания инсталлятора используйте скрипт `setup_script.iss` и программу Inno Setup.

## Требования

*   Windows 10/11
*   Интернет-соединение (для работы синтезатора речи)
